Blaster Wiring Guide (Physical Layout + Wire Planning)

This version is optimized for real-world assembly inside your blaster.

------------------------------------------------------------------------

🎯 Layout Strategy

Place components to minimize wire length:

-   NodeMCU: central
-   MAX98357A: near speaker
-   microSD: near side access (for card removal)

------------------------------------------------------------------------

🎨 Wire Color Convention (Recommended)

  Signal Type   Color
  ------------- --------
  5V (VU)       Red
  3.3V          Orange
  GND           Black
  SPI MOSI      Yellow
  SPI MISO      Green
  SPI SCK       Blue
  SPI CS        Purple
  I2S LRC       White
  I2S BCLK      Gray
  I2S DIN       Brown

------------------------------------------------------------------------

🎧 Audio Wiring (Keep VERY Short)

  NodeMCU   →   MAX98357A   Notes                     Wire Color   Length
  --------- --- ----------- ------------------------- ------------ ------------
  VU        →   VIN         5V for best audio power   Red          Short
  GND       →   GND         Common ground             Black        Short
  D4        →   LRC         Word select (WS)          White        Very short
  D2        →   BCLK        Bit clock                 Gray         Very short
  RX        →   DIN         Data in                   Brown        Very short

⚠️ Keep all I2S wires under ~2–3 inches if possible

------------------------------------------------------------------------

🔊 Speaker Wiring

  MAX98357A   →   Speaker   Wire
  ----------- --- --------- --------------
  SPK+        →   +         Twisted pair
  SPK-        →   -         Twisted pair

✔ Twist wires together to reduce noise

------------------------------------------------------------------------

💾 microSD Wiring (Moderate Length OK)

  NodeMCU   →   SD Module   Notes                     Wire
  --------- --- ----------- ------------------------- ------------
  3V3       →   VCC         3.3V ONLY (important)     Orange
  GND       →   GND         Common ground             Black
  D7        →   MOSI        SPI data out              Yellow
  D6        →   MISO        SPI data in               Green
  D5        →   SCK         SPI clock                 Blue
  D1        →   CS          Chip select               Purple

✔ Keep under ~6 inches if possible

------------------------------------------------------------------------

🎮 Button Wiring

  Button         Pin     GPIO   Notes
  -------------- ------- ------ -------------------------
  Trigger        D0      GPIO16 OK (simple input only)
  Reload         D3      GPIO0  ⚠ must stay HIGH at boot
  Next           TX      GPIO1  OK if not using serial prints heavily
  Previous       A0      ADC    Optional (analog ladder)
  Mode / Bank    D1      GPIO5  (if SD CS moved)

Wiring: Pin → Button → GND

🔌 Ladder Wiring

    3V3
     |
    [10k]  ← Pull-up
     |
     +------ A0 (NodeMCU)
     |
     +--[Button]--[1k]--- GND      → Next
     |
     +--[Button]--[2.2k]- GND      → Previous
     |
     +--[Button]--[4.7k]- GND      → Reload
     |
     +--[Button]--[10k]-- GND      → Bank


------------------------------------------------------------------------

⚡ Power Distribution

Recommended Layout:

-   Run a main ground line (black) across all components
-   Split power:
    -   5V → MAX98357A (short, thick wire if possible)
    -   3.3V → SD module

------------------------------------------------------------------------

🧠 Physical Mounting Tips

NodeMCU

-   Mount centrally
-   Leave USB accessible

MAX98357A

-   Place near speaker
-   Keep audio wires short
-   Add foam or padding to prevent vibration

microSD Module

-   Place near exterior for access
-   Secure firmly (cards can loosen)

------------------------------------------------------------------------

🔧 Strain Relief

-   Use hot glue or zip ties at connection points
-   Avoid stress on:
    -   RX (GPIO3)
    -   D8 (GPIO15)

------------------------------------------------------------------------

🔊 Noise Reduction Tips

-   Twist speaker wires
-   Keep I2S wires away from power lines
-   Avoid running audio wires parallel to SPI wires

------------------------------------------------------------------------

🧪 Final Assembly Checklist

-   All grounds connected
-   No long loose wires
-   SD card accessible
-   Speaker secure
-   No wires pulling on pins
